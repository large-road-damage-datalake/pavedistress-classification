# Abstract

PaveDistress Classification is a curated packaging of a pavement distress dataset released on Mendeley Data for reproducible benchmarking.

The source data contains 6032 high-resolution road-surface images (3854x2065) organized into Background, Crack, Other, Patch, and Pothole classes. Data were collected along the S315 highway in China using an inspection vehicle with a Basler line-scan camera and infrared laser-assisted lighting.

This package preserves publication metadata and adds generated statistics and visualization artifacts for the Large Road Damage DataLake.
