# PaveDistress

## Overview

PaveDistress is a task-specific package derived from the Mendeley Data dataset.

- Task: classification
- Images: 6032
- Annotations: 6032
- Classes: 5 (Background, Crack, Other, Patch, Pothole)
- Annotation format: image-level labels in class folders

This package contains high-resolution pavement images captured by an inspection vehicle along the S315 highway in China.
